# Clasificador Tributario · Sullair

App para clasificar automáticamente facturas de alquiler de equipos según jurisdicción tributaria (Añelo o Neuquén/Otros).

## Cómo usar

1. Subir el archivo Excel de facturas
2. Descargar el resultado clasificado

## Reglas de clasificación

- **Añelo**: coordenadas GPS ≤ 15 km del centro de Añelo, o localidades: Añelo, Bajo Añelo, Bajada de Añelo, BAN, Tratayen, Loma Campana
- **Neuquén/Otros**: todo lo demás
